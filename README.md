# 🗓️ Java Learning Plan (25 Days)

## ✅ WEEK 1: Fundamentals  
🎯 **Goal:** Understand syntax, variables, control flow  

### Day 1: Java Basics Marathon  
📘 `int`, `double`, `String`, `Scanner`, `System.out.print()`, `if-else`, `switch`, `for`, `while`, `do-while`  

🧠 **Problems**  
- Ask name and age, then print them.  
- Swap two numbers.  
- Print sum, difference, and product of two inputs.  
- Check if number is even or odd.  
- Grade checker (90 = A, 80 = B…)  
- Count from 1 to 10  
- Print even numbers up to N  

💡 **Hints**  
- Use `Scanner` for user input  
- `%` operator for checking even/odd  
- Loops: `for`, `while`, `do-while`  
- Conditionals: `if`, `else`, `switch-case`  

🛠 **Project: All-in-One Console App**  
- Greet user, ask age  
- Menu: Calculator, Grader Average, Loop Printer  
- Use loops, conditionals, and input/output together  
🎯 Clue: Make it interactive — menu system using `while (true)` and `switch`  

### Day 2: Data Types, Type Casting, Math Logic  
📘 `char`, `boolean`, type casting, `Math` class  

🧠 **Problems**  
- Convert `double` to `int`  
- Boolean checker for adult age (18+)  
- Store and display first initial using `char`  
- Find max of 3 numbers using `if-else`  
- Create a simple login with hardcoded username/password  

🛠 **Project: BMI Calculator + Age Checker**  
- Input height and weight, compute BMI  
- Add conditional: Check if underweight, normal, overweight  
🎯 Clue: Use `weight / (height * height)` with `if-else`  

### Day 3: Arrays (1D)  
📘 `int[]`, loops with arrays  

🧠 **Problems**  
- Store 5 grades and print average  
- Find max and min in array  
- Count how many are above average  

🛠 **Project: Student Scores Manager**  
- Input and analyze scores of 5 students  

### Day 4: Array Operations  
📘 Reversing, sorting (bubble sort), searching  

🧠 **Problems**  
- Reverse an array  
- Bubble sort an array  
- Search for a number  

🛠 **Project: Leaderboard Ranker**  
- Sort and display top 3 scorers  

### Day 5: Strings Basics  
📘 `.length()`, `.charAt()`, `.substring()`  

🧠 **Problems**  
- Reverse a string  
- Count vowels  
- Check if palindrome  

🛠 **Project: Username Validator**  
- Min 6 chars, no spaces, lowercase only  

### Day 6: String Functions  
📘 `.equals()`, `.toUpperCase()`, `.replace()`  

🧠 **Problems**  
- Replace all vowels with `*`  
- Convert to uppercase  
- Case-insensitive comparison  

🛠 **Project: Simple Login System**  
- Hardcoded credentials + retry loop  

### Day 7: Methods (Functions)  
📘 `void`, `return`, parameters  

🧠 **Problems**  
- Method to add 2 numbers  
- Method to check even/odd  
- Method that returns factorial  

🛠 **Project: Math Library**  
- Methods: square, cube, power, etc.
